# Veterinaria Animales Fantásticos

Este es un proyecto de aplicación de Android para gestionar una clínica veterinaria. La aplicación permite administrar mascotas, dueños, veterinarios y consultas.

## Características

*   **Dashboard principal**: Muestra un resumen del estado de la veterinaria.
*   **Gestión de Mascotas**: Permite añadir, editar y eliminar registros de mascotas.
*   **Gestión de Dueños**: Permite añadir, editar y eliminar registros de dueños.
*   **Gestión de Veterinarios**: Permite añadir, editar y eliminar registros de veterinarios.
*   **Agenda de Consultas**: Permite programar y gestionar las consultas.

## Tecnologías Utilizadas

*   Kotlin
*   Jetpack Compose
*   ViewModel
*   Navigation Component
